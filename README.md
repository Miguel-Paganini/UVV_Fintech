# UVV_Fintech

Para criar o banco localmente use o seguinte comando no console do gerenciador de pacotes
``` bash
PM> Update-Database
```

Para Verificar se o banco foi criado corretamente use o Pesquisador de Objetos do SQL Server.

Toda vez que fizer alguma alteração no codigo, por menor que seja realize o commit dessa alteração com uma mensagem clara explicando o que foi alterado
